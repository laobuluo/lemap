# LeMap - WordPress文章地图插件

一个简单而优雅的WordPress文章地图/存档插件，按年份和月份组织展示所有文章。

## 插件介绍

LeMap插件可以在您的WordPress网站上生成一个美观的文章地图页面，帮助访客快速浏览和查找所有文章内容。

### 主要特点

- 自动生成静态HTML文章地图
- 按年份和月份层级展示文章
- 自动更新，无需手动维护
- 支持自定义页面标题
- 响应式设计，移动端友好
- 简洁优雅的界面设计

## 安装说明

1. 下载插件压缩包
2. 在WordPress后台点击"插件 > 安装插件"
3. 点击"上传插件"按钮
4. 选择下载的压缩包并安装
5. 安装完成后启用插件

## 使用方法

1. 插件启用后会自动在网站根目录生成文章地图页面（默认为 lemap.html）
2. 在WordPress后台"设置 > LeMap"中可以：
   - 启用/禁用文章地图功能
   - 自定义文章地图文件名
   - 自定义页面标题

## 常见问题

**Q: 文章地图页面在哪里查看？**  
A: 默认访问网站根目录下的 lemap.html，例如：`https://your-site.com/lemap.html`

**Q: 文章更新后多久同步到地图页面？**  
A: 文章发布、更新、删除时会自动更新地图页面，无需手动操作。

**Q: 如何修改文章地图的样式？**  
A: 插件提供了简洁优雅的默认样式，暂不支持在后台直接修改样式。如需自定义，请修改插件源码中的CSS部分。
